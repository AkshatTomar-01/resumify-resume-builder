# Resumify – AI Resume Builder 🧠 🧾

**Resumify** is a full-stack SaaS application that empowers users to build professional resumes with the help of AI.  
It offers a live resume editor, smart AI improvements, customizable templates, and easy sharing and download options.

---

## 🚀 Features

- 📄 **AI-powered resume generation & enhancement**  
  Auto-suggests professional text for summaries and job descriptions.

- ✍️ **Live resume editor**  
  See real-time updates as you type.

- 🎨 **Multiple templates & themes**  
  Choose from ready-to-use templates and colour options.

- 🔐 **User authentication & dashboard**  
  Users can sign up, log in, and manage multiple resumes.

- 📤 **Export & share**  
  Download as PDF or share via a secure live link.

---
